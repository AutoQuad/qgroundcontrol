RC-Adjustable Parameters

This feature lets you adjust certain parameters while in-flight. For example to tune attitude settings for PID or Quatos controller.

Be very careful when using this option! Use the setup interface with RC Telemetry active to make sure you are getting the desired result before flying!

You can control the amount of adjustment applied by setting the Scale parameter described below, and also using your RC Tx to set end-points for the channels being used. For example by setting a small channel output range (end-points close together), you can get very precise adjustment of a parameter over the full physical throw of your knob/slider control. Conversely, setting a large output range will let you make large adjustments quickly.

Scale

This is the multiplier applied to the radio channel value before adjustment of the parameter value. Best explained as a formula where:

Pf is the final parameter value.
Po is the original value from settings.
Rv is the current radio channel value.

Pf = Po + (Po * Rv * Scale)

Example, let's say you are adjusting Tilt Rate D starting with default value of 34600, your radio channel value is -250, and the scale is default of 0.002:

Pf = 34600 + ( 34600 * (-250) * 0.002 ) = 17300

You can see the result is exactly half of the original value. Another way to look at the Scale setting is as a percentage, so 0.002 means scale the value 0.2% for every radio channel step. 250 * 0.2% = 50%, which is our result in the example.

You can select a Scale value of up to 0.0255, which would be 2.55% for every radio value step (very large changes are possible, be careful!).
